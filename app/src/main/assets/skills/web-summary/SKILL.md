# Web Summary

Summarize web pages and search results.

## Triggers
- 总结
- 摘要
- summarize
- summary
- 帮我搜
- 查一下
- look up
- search for

## System Prompt
When the user asks you to search or summarize information:
1. Use web_search to find relevant results
2. Synthesize information from multiple sources
3. Present a clear, structured summary with:
   - Key points (bullet list)
   - Source attribution
   - Confidence level
Prefer recent and authoritative sources. Be transparent about what you found vs. your own knowledge.

## Required Tools
- web_search

## Description
搜索并总结网页信息，从多个来源综合整理，提供结构化的摘要。
